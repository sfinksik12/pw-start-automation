# DocSearch Modal — документация фичи

## Описание

Модальное окно поиска по документации. Пользователь может быстро найти нужный раздел без ручной навигации по меню.

---

## Сценарии использования

### 1. Поиск по документации

Пользователь вводит запрос в поле поиска и получает список релевантных страниц. Результаты отображаются в реальном времени (автодополнение). Максимальная длина запроса — 64 символа.

### 2. Быстрый доступ к недавним поискам

При открытии модалки отображается блок «Recent» — недавние поисковые запросы. Пользователь может повторно выбрать один из них без повторного ввода.

### 3. Сохранение в избранное

У каждого результата есть кнопка «сохранить» (звёздочка). Пользователь может добавить часто используемые страницы в избранное для быстрого доступа.

### 4. Удаление из истории

Кнопка «удалить» (крестик) у результата позволяет убрать его из списка недавних поисков.

### 5. Закрытие модалки

Отмена — кнопка «Cancel» или клавиша Escape. Закрытие без перехода на страницу.

---

## Пользовательский поток

1. Клик по кнопке «Search» в хедере → открывается модалка
2. Ввод запроса → поиск, отображение результатов
3. Выбор результата (клик или Enter) → переход на страницу
4. Или Cancel / Escape → закрытие без перехода

---

## Критерии приёмки

| #   | Критерий                                        | Статус |
| --- | ----------------------------------------------- | ------ |
| 1   | Модалка открывается по клику на Search в хедере | ✓      |
| 2   | Поле ввода принимает до 64 символов             | ✓      |
| 3   | Результаты отображаются при вводе               | ✓      |
| 4   | Блок Recent показывает недавние поиски          | ✓      |
| 5   | Можно сохранить результат в избранное           | ✓      |
| 6   | Можно удалить результат из истории              | ✓      |
| 7   | Поддержка клавиатуры: Enter / ↑↓ / Escape       | ✓      |
| 8   | Кнопка Cancel закрывает модалку                 | ✓      |

---

## Технические ограничения

- Поиск выполняется через внешний сервис Algolia
- Recent и Favorites хранятся в localStorage браузера (привязаны к устройству)
